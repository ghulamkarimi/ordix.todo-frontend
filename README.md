#  Todo-Ordix Frontend

[![Frontend CI](https://github.com/ghulamkarimi/ordix.todo-frontend/actions/workflows/frontend-ci.yml/badge.svg)](https://github.com/ghulamkarimi/ordix.todo-frontend/actions)

Dieses Projekt ist das Frontend für die Todo-App, gebaut mit **React**, **TypeScript** und **Vite**.

##  Features
* **CI/CD integriert:** Jeder Push wird automatisch via GitHub Actions auf Build-Fehler geprüft.
* **Redux Toolkit:** State Management mit robustem Error-Handling.
* **Typ-Sicherheit:** Vollständige TypeScript-Unterstützung für API-Responses.

## Tech Stack
* **Framework:** React + Vite
* **Sprache:** TypeScript
* **State Management:** Redux Toolkit (@reduxjs/toolkit)
* **API-Anfragen:** Axios
